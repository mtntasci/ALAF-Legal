# ALAF Legal – AI-Powered Legal Draft Generator (MVP Demo)

## Overview

**ALAF Legal** is a mobile-first web application that allows users to upload legal documents and instantly generate structured legal drafts such as defense statements, objections, or appeals.

Unlike generic AI tools, ALAF Legal does **not rely on user prompts**. Instead, it guides the user through a structured flow and produces ready-to-use legal documents.

> Upload your file → Answer a few questions → Get your legal draft.

---

## Key Features

* 📄 **Document Upload (PDF/Text)**
* 🧭 **Guided Flow (No Prompt Required)**
* ⚖️ **Legal Draft Generation (Structured Output)**
* 🧠 **AI Simulation (Demo Mode)**
* 📱 **Mobile-First Design**
* 🔒 **Privacy-Oriented (No file storage in MVP)**

---

## Demo Flow

1. Upload a legal document
2. Select the action (Defense / Objection / Appeal)
3. Provide optional input
4. System processes the data
5. View generated legal draft
6. Download result
